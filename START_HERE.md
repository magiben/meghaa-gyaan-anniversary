# 🎉 Anniversary Website - Start Here!

## 🚀 Quick Start (Copy & Paste)

```bash
npm install
npm run setup
npm run dev
```

Then open: **http://localhost:3000**

---

## 📝 How to Add Your Content

1. **Click** the "Edit Our Story ✨" button (bottom right)
2. **Upload** your images and videos (max 10MB each)
3. **Edit** all the text you want to change
4. **Click** the "Save Changes" button at the bottom
5. **Wait** for "✓ Saved successfully!" message

---

## 🌐 How to Share

Once you've saved your changes:

1. **Your content is saved** in `public/data/site-data.json`
2. **Anyone who visits** your website will see it
3. **Share the link** with anyone you want
4. **They'll see** all your images, videos, and text

## ⚠️ Important Notes

### File Sizes
- **Maximum**: 10MB per file
- **Recommended**: Images < 2MB, Videos < 5MB
- **Warning**: Large files take longer to save

### Saving Changes
- Changes save **locally** as you edit (instant)
- Changes save **to server** when you click "Save Changes"
- **Others won't see** changes until you click "Save Changes"

### Sharing
- After clicking "Save Changes", **anyone** can see your content
- Just share your website URL
- They don't need to do anything special

## 🔧 Troubleshooting

### "Failed to save" error?
```bash
npm run setup
```

### Want to verify everything?
```bash
npm run verify
```

### Need to backup your data?
```bash
cp public/data/site-data.json backup.json
```
