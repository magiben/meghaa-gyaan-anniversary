FIRST TIME SETUP:
-----------------
1. Run: npm install
2. Run: npm run setup
3. Run: npm run dev
4. Open: http://localhost:3000

UPLOADING YOUR CONTENT:
-----------------------
1. Click "Edit Our Story ✨" button on the website
2. Upload images (max 10MB each)
3. Upload videos (max 10MB each)
4. Edit all text fields
5. Click "Save Changes" button at the bottom
6. Wait for "✓ Saved successfully!" message

SHARING WITH OTHERS:
--------------------
Once you save changes, the data is stored in:
  public/data/site-data.json

Anyone who visits your website URL will see:
  ✓ All your uploaded images
  ✓ All your uploaded videos
  ✓ All your text changes

IMPORTANT NOTES:
----------------
• File size limit: 10MB per file
• Best performance: Keep images under 2MB, videos under 5MB
• Data is saved on the server, not in browser
• Share your website URL with anyone to show them your content

TROUBLESHOOTING:
----------------
If "Save Changes" fails:
  1. Make sure public/data folder exists (run: npm run setup)
  2. Refresh the page and try again
  3. Check that files are under 10MB
